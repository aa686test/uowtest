Name Web Attacks
Man in the Middle Attacks

source: https://www.thesslstore.com/blog/man-in-the-middle-attack-2 		In Man in the Middle attack an attacker intercepts the communication between two parties and can potentially read, modify, or even insert new information without the knowledge or consent of the communicating parties.Once the attackers interrupt the traffic, they can filter and steal data.MITM attacks can be carried out in various ways, such as by compromising Wi-Fi networks, infiltrating routers, or utilizing phishing techniques to divert traffic. Once the attacker gains control over the communication channel, they can collect sensitive information, inject malicious code, or even impersonate one of the parties involved.

Phishing Attack

source: https://www.cloudflare.com/en-gb/learning/access-management/phishing-attack/ 		Phishing attacks are a type of cyber attack where attackers try to trick individuals into providing personal information, such as passwords, credit card details, or login credentials, by pretending to be a legitimate entity. They often use social engineering tactics to create a sense of urgency or fear, making victims more likely to fall for the scam. These attacks can be carried out through emails, text messages, or even phone calls.

Ref : https://www.fortinet.com/resources/cyberglossary/types-of-cyber-attacks
DoS and DDoS Attacks

source: https://www.wallarm.com/what/dos-denial-of-service-attack 		A denial-of-service (DoS) attack and a distributed denial-of-service (DDoS) attack are both types of cyber attacks that aim to disrupt the normal functioning of a system or websiteA denial-of-service (DoS) attack is designed to overwhelm the resources of a system to the point where it is unable to reply to legitimate service requests. A distributed denial-of-service (DDoS) attack is similar in that it also seeks to drain the resources of a system. A DDoS attack is initiated by a vast array of malware-infected host machines controlled by the attacker.

Ref : https://www.fortinet.com/resources/cyberglossary/types-of-cyber-attacks
Cryptojacking

source:https://www.bitpanda.com/academy/en/lessons/what-is-cryptojacking-and-how-can-i-prevent-it/ 		Cryptojacking is unauthorized use of someone's computer processing power to mine cryptocurrencies. This is typically done without the person's knowledge or consent. Cybercriminals use various methods to perform cryptojacking, such as infecting websites with malicious code or distributing infected files via phishing emails.

Ref :https://www.aura.com/learn/types-of-cyber-attacks
SQL Injection

source: https://spanning.com/blog/sql-injection-attacks-web-based-application-security-part-4/ 		An SQL injection attack is a type of web application vulnerability where an attacker manipulates a website's input fields, particularly those that interact with a database using SQL queries. By injecting malicious SQL code into the vulnerable input field, the attacker can trick the server into executing unintended commands.This attack can occur when the website fails to properly validate or sanitize user input, allowing the attacker to insert their own SQL commands. The injected code can alter the intended query's logic or reveal sensitive information from the database.

Ref : https://www.simplilearn.com/tutorials/cyber-security-tutorial/types-of-cyber-attacks
Watering Hole Attack

source: https://us.norton.com/blog/emerging-threats/what-is-a-watering-hole-attack 		Watering hole attacks are a type of targeted attack where the attacker identifies websites commonly visited by a specific group of individuals or organization. The goal is to infect these websites with malware, which in turn infects the systems of the targeted users when they visit those compromised websites.Once an attacker compromises a website, they inject malicious code or malware into it, typically in the form of exploit kits. These exploit kits exploit vulnerabilities in the users' web browsers or plugins, allowing the malware to be downloaded and executed on their systems.

Ref :https://www.simplilearn.com/tutorials/cyber-security-tutorial/types-of-cyber-attacks
Zero Day Exploit

source: https://www.darkreading.com/threat-intelligence/zero-day-exploit-use-exploded-in-2021 		Zero-day vulnerabilities pose a significant threat because attackers exploit them before a patch or solution is in place. Zero-day exploits are particularly challenging because they are unknown to the software vendor and, therefore, aren't patched.Zero day exploits are particularly concerning because they can be used to breach systems and compromise data or control over a system without any prior knowledge or defense against it. Zero-day vulnerability threat detection requires constant awareness.

By doing so, they can communicate covertly with command and control servers without raising suspicion.

RootKits

source: https://www.pandasecurity.com/en/mediacenter/security/rootkits-the-invisible-threat-2/ 		Rootkits are a particularly stealthy type of malware that can grant unauthorized users complete control and administrator-level access to a targeted system. They are designed to hide deep within the operating system, making them difficult to detect even for advanced security measures.

URL Manipulation

source: https://www.consuro.com/url-manipulation-and-what-to-do-about-it/ 		URL manipulation is a technique employed by hackers to manipulate the parameters within a URL address. By altering these parameters, they can trick users into visiting malicious websites, redirect them to phishing sites, or attempt to download malware onto their devices. For example, many people use URL shorteners to help remember long web addresses or specific pages. If hackers “poison” that shortened URL, they can send you to a phishing site designed to steal your personal information.
